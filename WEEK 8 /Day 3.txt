ans) in inserting sorting iniially we take only one elements then compare the first with second elements and sort the 2 elements again we take 3 elements and iniially looping the back form and compare the elements

class Solution {
  public:
    void insertionSort(vector<int>& arr) {
        int n = arr.size();
        for(int i = 0 ; i< n ; i++){
            int currelement = arr[i];
            int j = i-1 ;
            while(j>=0 && arr[j]>=currelement){
                arr[j+1] = arr[j];
                j--;
            } arr[j+1] = currelement;
        }
    }
};

Q) what is bubble sort ?
ans)  To sort a data set using bubble sort algorithm, follow the below steps:

->Start by comparing the first two elements. If they are in the wrong order, swap them.
->Continue this process for all elements moving from left to right. After the first pass, the largest element will be at the end.
->In the next pass, skip the last element since it’s already sorted and repeat the above steps. The ->second-largest element will move to the second-last position.
->Repeat the steps until the entire array is sorted.


code: 
void bubbleSort(vector<int>& v) {
    int n = v.size();
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - i - 1; j++) {
            if (v[j] > v[j + 1])
                swap(v[j], v[j + 1]);
        }
    }
}


Q) what is selection sort ?
ans) 
It sorts an array by repeatedly selecting the smallest (or largest) element from the unsorted portion and swapping it with the first unsorted element. This process continues until the entire array is sorted.

->First we find the smallest element and swap it with the first element. This way we get the smallest element at its correct position.
->Then we find the smallest among remaining elements (or second smallest) and swap it with the second element.
-> We keep doing this until we get all elements moved to correct position.

void selectionSort(vector<int> &arr) {
    int n = arr.size();
    for (int i = 0; i < n - 1; ++i) {
        int min_idx = i;
        for (int j = i + 1; j < n; ++j) {
            if (arr[j] < arr[min_idx]) {
                min_idx = j; 
            }
        }
        swap(arr[i], arr[min_idx]);
    }
}

Q) " leetcode 78 " subsets ?
ans) in this question we used backtracking 

// we are always take index when we passed the index 

class Solution {
public:
    void getallsubsets(vector<int>& nums, vector<int>& ans, int i, vector<vector<int>>& allsubsets) {
        if (i == nums.size()) {
            allsubsets.push_back(ans); 
            return;
        } 
        ans.push_back(nums[i]);
        getallsubsets(nums, ans, i + 1, allsubsets);

        ans.pop_back();
        getallsubsets(nums, ans, i + 1, allsubsets);
    }

    vector<vector<int>> subsets(vector<int>& nums) {
        vector<vector<int>> allsubsets;
        vector<int> ans;
        getallsubsets(nums, ans, 0, allsubsets);
        return allsubsets; 
    }
};

                      How the Function Works

->Base Case: Stop when i == nums.size(), store the subset.
->Always Move Forward (i+1) to avoid reusing previous numbers.
   
                      Backtracking Steps:

->Include the number and recurse.
->Remove (pop_back) the number to backtrack.
->Exclude the number and recurse again.

                      Key Observations

.)Each Recursive Call Creates a New Branch (like a tree).
.)First Include, Then Exclude to ensure correct subset generation.
.)Order of Calls Matters to avoid duplicate subsets.
.)Backtracking efficiently explores all possible subsets.
