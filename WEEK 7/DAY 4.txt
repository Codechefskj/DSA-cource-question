Q) leetcode 31 next permutation ?
ans) 

In this question we check left hand side is sorted or not when no are break the sorted the we swap the no which is just greater than those no and then reverse the left over no 

class Solution {
    public void nextPermutation(int[] nums) {
        int i = nums.length - 2; 
        while (i >= 0 && nums[i] >= nums[i + 1]) {
            i--;
        }

        if (i >= 0) {
            int j = nums.length - 1;
            while (nums[j] <= nums[i]) {
                j--;
            }
            swap(nums, i, j);
        }
        reverse(nums, i + 1);
        }

    private void reverse(int[] nums, int start) {
        int i = start, j = nums.length - 1;
        while (i < j) {
            swap(nums, i, j);
            i++;
            j--;
        }
    }

    private void swap(int[] nums, int i, int j) {
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}
