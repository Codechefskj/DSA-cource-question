tree terminalogy (in case of rooted) : 
Node: A point that stores data.
Root: Top node (starting point).
Edge: Line connecting two nodes.
Parent: Node above another node.
Child: Node below a parent.
Siblings: Nodes with same parent.
Leaf Node: Node with no children (last node).
Internal Node: Node that has children.
Degree: Number of children a node has.
Path: Route from one node to another.
Level: Position of node from top.
Depth: Distance from root to node.
Height: Distance from node to bottom.
Ancestor: Nodes above a node.
Descendant: Nodes below a node.
Subtree: Small tree inside a tree.
Forest: Group of separate trees.
